# 🖥️ Windows 11 RDP

Access a **Windows 11 Remote Desktop** with **free 4GB/s internet speed** instantly!  
This project provides a quick and reliable way to connect to a powerful remote environment for development, testing, or personal use. 🚀

---

## 📜 Description

The **Remote Desktop Protocol (RDP)** project allows users to connect to a **Windows 11** virtual machine in the cloud, offering:

- ⚡ **4GB/s Internet Speed**  
- 💻 **Windows 11 OS**  
- 🌍 **Free and Easy Remote Access**  
- 🔒 **Secure and Fast Connection**

## 🧰 Features

- 💨 Blazing-fast internet connection

## 💽 Setup Steps

### Tailscale Setup

Tailscale : https://tailscale.com/

1.Open the Tailscale Website and Sign in with using GITHUB

2.Install Tailscale App [where you want to install]

3.Go to the Tailscale App Sign in with the Account and start it

4.Go to the Tailscale Website and Click the Device model [ it will Your Phone model will show when you will open the vpn mean Tailscale]

5.Now,Go to Settings and select Keys

6.Generate a Auth and Api key and save it in a Safe place.

### Main Setup

Github Repository : https://github.com/tahseenmahmud09032015/Tailscale-Windows-11-RDP/

1.Go to Aution Tab

2.Select the Windows 11 RDP [ Tailscale + RustDesk ]

3.Click Run WorkFlow and Fill the Email box,Tailscale Auth Key Box and the Tailscale Api key Box

4.Then Click Run workflow and Wait 2 MIN

5.Now Go to Tailscale App and click Bullet

6.Copy the IP and paste the IP on the Remote Desktop or any Remote Desktop App.
