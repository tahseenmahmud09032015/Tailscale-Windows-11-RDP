## 💽 Steps

1.Sign in on Tailscale https://tailscale.com/ [ Sign in with Github ]

2.Install Tailscale App on Phone or Computer [ Where you will use ]

3.Go To the Tailscale Website and Select your Phone Model name phone name 

4.Then go to settings and open key tab

5.Generate a Auth and Api key

6.Come to Github and go to Action tab

7.Click the Windows RDP Workflow

8.Click Run Workflow

9.Enter Your Email,Tailscale Auth and Api key and click Run Workflow

Yeah,Now your Rdp is Ready.Go to Tailscale app and open the Bullet File and copy the IP
and Paste it in your Remote Desktop APP.Make sure open the VPN of Tailscale
